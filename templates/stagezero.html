Stage 0: the main ideas

How the Web works (roughly)

There are about 30 billion pages on the web.These pages are stored on servers and read by a computer's browser that accesses them through the Internet (the largest computer network on Earth).

What HTML is (kinda)

HTML stands for Hyper Text Markup Language.
It is a languge that uses tags to explain to internet browsers
what a webpage contains and to some extent how it is to be displayed.
