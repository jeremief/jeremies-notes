Meet JavaScript

JavaScript is THE programming language of the web. There are two was of adding it: firstly into the html through a script tag <script></script> or in a separate .js file. Like here.

Here are some new concepts:
  • Declaring variables: the syntax is the same for strings and numbers. var declares a new variable for all data types: var variableName = someValue.
  • Replacing strings: follow the format string.replace("old","new")
You can use the console of the webbrowser to debug your Javascript:
<script type="text/javascript">
var firstName = "Jeremie";
console.log (firstName);
</script>
